Set de imágenes de Dados de Seis Caras
======================================

Hay dos subsets dentro de este paquete:

	* Un set de dados con resolución 300 dpi
	* Un set de dados con resolución  72 dpi

Ambos sets están formados por seis imágenes, una por cada cara de un dado, y se han generado en 5 colores: verde, azul, gris, rosa y rojo.
